# dotfiles

My configs, featuring bash with [liquidprompt], tmux, vim, [fzf], [dotbot] and others.

# Installation

Clone repo and install using [dotbot]:

    cd && git clone https://github.com/akselsjogren/dotfiles.git && dotfiles/install
